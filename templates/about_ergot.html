{{ _("Disease Overview") }}

{{ _("Understanding Ergot") }}

{{ _("Ergot is a serious fungal disease caused by Claviceps purpurea and related species. The fungus infects the flowers of cereals and grasses, replacing developing grains with dark, elongated structures called sclerotia. These sclerotia contain toxic alkaloids that pose significant risks to human and animal health.") }}

{{ _("The disease has been documented throughout history, responsible for numerous epidemics of ergotism in humans and livestock. Modern agricultural practices and regulatory standards have significantly reduced these risks, though ergot remains an important concern in crop production.") }}

{{ _("Primary hosts:") }} {{ _("Rye, wheat, barley, oats, triticale, and various wild and cultivated grasses including pearl millet.") }}

{{ _("Scientific Classification") }}

{{ _("Kingdom:") }} {{ _("Fungi") }}
{{ _("Phylum:") }} {{ _("Ascomycota") }}
{{ _("Class:") }} {{ _("Sordariomycetes") }}
{{ _("Order:") }} {{ _("Hypocreales") }}
{{ _("Family:") }} {{ _("Clavicipitaceae") }}
{{ _("Genus:") }} {{ _("Claviceps") }}

{{ _("Geographic Distribution") }}

{{ _("Ergot occurs worldwide in temperate and subtropical regions. Most prevalent in areas with cool, moist conditions during the flowering period of susceptible crops. Major concerns exist in Europe, North America, parts of Asia, and temperate regions of South America and Africa.") }}

{{ _("Disease Cycle & Infection Process") }}

{{ _("Infection Mechanism") }}

{{ _("Ergot follows a dual reproductive cycle. Sclerotia survive in soil or grain, germinate in spring, and release ascospores that infect flowers.") }}

  • {{ _("Primary infection:") }} {{ _("Wind-borne ascospores infect stigmas of open flowers.") }}
  • {{ _("Honeydew stage:") }} {{ _("Infected florets ooze sugary liquid containing conidia.") }}
  • {{ _("Secondary spread:") }} {{ _("Conidia spread via insects, rain, or contact.") }}
  • {{ _("Sclerotial stage:") }} {{ _("Ovaries turn into dark, hard sclerotia.") }}
  • {{ _("Overwintering:") }} {{ _("Sclerotia persist in soil and germinate next season.") }}
1

{{ _("Early Stage: Honeydew") }}

{{ _("Infected florets exude sweet, sticky honeydew containing conidia. This stage is critical for disease spread through insects and splashing rain.") }}

2

{{ _("Mid Stage: Sclerotia Development") }}

{{ _("Honeydew production ceases as the fungus begins forming sclerotia. Purple-black fungal tissue becomes visible, gradually hardening.") }}

3

{{ _("Late Stage: Mature Sclerotia") }}

{{ _("Hard, dark sclerotia fully replace the grain. These structures contain concentrated alkaloids and fall to the ground or are harvested with grain.") }}

{{ _("Favorable Environmental Conditions") }}

{{ _("Temperature Requirements") }}

{{ _("Optimal development occurs at 15-20 degrees Celsius (59-68 degrees Fahrenheit). Cool temperatures during flowering significantly increase infection rates. Temperatures above 28 degrees Celsius (82 degrees Fahrenheit) generally suppress disease development.") }}

{{ _("Moisture & Humidity") }}

{{ _("High humidity (greater than 70%%) and frequent rainfall or dew during flowering promote spore germination and honeydew production. Extended wet periods coinciding with anthesis create ideal infection conditions.") }}

{{ _("Host Susceptibility") }}

{{ _("Open-pollinated crops (especially rye) are most susceptible. Hybrid varieties with better floret closure show reduced infection. Prolonged flowering periods increase exposure to inoculum.") }}

{{ _("Crop Management Factors") }}

{{ _("Dense canopies with poor air circulation, excessive nitrogen fertilization promoting late flowering, and proximity to contaminated fields or wild grass hosts increase disease risk.") }}

{{ _("Health & Economic Impacts") }}

{{ _("Human Health") }}

{{ _("Consumption of ergot-contaminated grain causes ergotism, presenting in two forms:") }}

  • {{ _("Gangrenous ergotism: vasoconstriction leading to gangrene, blackened extremities") }}
  • {{ _("Convulsive ergotism: neurological symptoms including seizures, hallucinations, psychosis") }}
  • {{ _("Chronic effects: reproductive problems, immune suppression") }}
  • {{ _("Historical epidemics: St. Anthony Fire in medieval Europe") }}

{{ _("Livestock Impact") }}

{{ _("Ergot alkaloids severely affect grazing animals and livestock fed contaminated feed:") }}

  • {{ _("Reduced feed intake and weight loss") }}
  • {{ _("Reproductive failures: abortions, reduced fertility, agalactia") }}
  • {{ _("Gangrene of extremities (tail, ears, hooves)") }}
  • {{ _("Hyperthermia and heat stress intolerance") }}
  • {{ _("Death in severe cases") }}

{{ _("Economic Consequences") }}

{{ _("Ergot contamination has substantial economic implications:") }}

  • {{ _("Grain rejection at greater than 0.05%% contamination (EU standard)") }}
  • {{ _("Price penalties for contaminated lots") }}
  • {{ _("Complete loss of feed and food value") }}
  • {{ _("Cleaning and sorting costs") }}
  • {{ _("Market access restrictions") }}
  • {{ _("Reduced crop yields") }}

{{ _("Identification & Symptoms") }}

{{ _("Field Recognition") }}

{{ _("Early Detection Signs") }}

{{ _("Look for glistening honeydew droplets on flowers during anthesis. The sticky substance may attract insects and may have a slightly sweet odor. Individual florets appear discolored or damaged.") }}

{{ _("Mature Sclerotia Characteristics") }}

{{ _("Sclerotia are dark purple to black, hard, elongated structures (5-50mm long) protruding from heads. They are significantly larger than normal grain, curved or horn-shaped, and have a rough exterior surface.") }}

{{ _("Distribution Patterns") }}

{{ _("Infection typically appears scattered throughout fields but may be concentrated in low-lying areas with poor drainage, field edges near wild grasses, or areas with extended flowering due to uneven maturity.") }}

{{ _("Integrated Management Strategies") }}

{{ _("Cultural Practices") }}

  • {{ _("Deep plowing to bury sclerotia (greater than 10cm depth)") }}
  • {{ _("Crop rotation with non-susceptible crops") }}
  • {{ _("Delayed planting to avoid high-risk flowering periods") }}
  • {{ _("Mow wild grass hosts near fields before flowering") }}
  • {{ _("Ensure uniform crop maturity through proper spacing") }}
  • {{ _("Avoid excessive nitrogen late in season") }}

{{ _("Resistant Varieties") }}

  • {{ _("Use hybrid varieties with better floret closure") }}
  • {{ _("Select varieties with shorter flowering periods") }}
  • {{ _("Choose cultivars bred for ergot resistance") }}
  • {{ _("Consider male-sterile lines carefully (higher risk)") }}
  • {{ _("Maintain variety purity to ensure uniform flowering") }}

{{ _("Chemical Control") }}

  • {{ _("Fungicide application at early flowering (10%% anthesis)") }}
  • {{ _("Repeat applications during extended flowering") }}
  • {{ _("Products: azoxystrobin, prothioconazole, tebuconazole") }}
  • {{ _("Timing is critical - preventive rather than curative") }}
  • {{ _("Monitor weather forecasts for high-risk periods") }}

{{ _("Seed & Grain Management") }}

  • {{ _("Use certified, ergot-free seed") }}
  • {{ _("Clean harvesting equipment between fields") }}
  • {{ _("Segregate contaminated lots immediately") }}
  • {{ _("Test grain for ergot content before storage/sale") }}
  • {{ _("Never use contaminated grain for seed") }}

{{ _("Mechanical Cleaning") }}

  • {{ _("Size-based separation (sclerotia larger than grain)") }}
  • {{ _("Gravity tables exploit density differences") }}
  • {{ _("Color sorting technology for large operations") }}
  • {{ _("Multiple cleaning passes for thorough removal") }}
  • {{ _("Regular equipment calibration and maintenance") }}

{{ _("Monitoring & Scouting") }}

  • {{ _("Scout fields weekly during flowering") }}
  • {{ _("Record infection levels and locations") }}
  • {{ _("Monitor wild grass hosts in field margins") }}
  • {{ _("Track weather conditions during critical periods") }}
  • {{ _("Maintain field history records") }}
